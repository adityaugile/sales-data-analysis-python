# Sales Data Analysis Using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a Superstore Sales dataset using Python.
The objective is to analyze sales trends, understand category and regional performance, and extract actionable business insights.

## Dataset
- Records: 9,800
- Features: 18 columns including Order Date, Category, Region, Sales, and Customer details
- Source: Superstore Sales Dataset

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Key Analysis Performed
- Data cleaning and inspection
- Date parsing and feature engineering
- Monthly sales trend analysis
- Category-wise and region-wise sales analysis
- Identification of high-performing segments

## Key Insights
- Sales exhibit seasonality with noticeable peaks during certain periods.
- Technology is the top-performing category in terms of sales.
- West and East regions contribute the highest revenue.
- A limited number of products account for a large portion of sales.
